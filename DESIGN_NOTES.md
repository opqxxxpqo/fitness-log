# 设计稿解析摘要

来源：`ACN1~ACN4 _ _.html`（Figma 导出）

## 设计系统

### 颜色（直接从设计稿提取）
| Token | Hex | 用途 |
|-------|-----|------|
| `bg`        | `#F2F0EC` | 主背景（暖灰白 / 纸质感） |
| `surface`   | `#E7E4DE` | 卡片次要表面、分割块 |
| `ink`       | `#0A0A0A` | 主文字 |
| `ink-2`     | `#1A1A1A` | 副文字 |
| `ink-3`     | `#222222` | 三级文字 |
| `ink-55`    | `rgba(10,10,10,0.55)` | 灰文字 |
| `ink-12`    | `rgba(10,10,10,0.12)` | 描边 / 分割线 |
| `accent`    | `#FF3B1F` | 强调红橙（CTA、未完成状态、高光数据） |
| `white`     | `#FFFFFF` |  |

### 字体
- `Doto` — 大号像素风数字（项目数 / 重量 / 计时 / 数据）
- `JetBrains Mono` — 等宽，用于代码风的 `// 标签`、单位、tab 文字
- `Noto Sans SC` — 中文正文
- `VT323` — 可能用于状态栏数字

### 排版
- 字号阶梯：`8 / 9 / 10 / 11 / 13 / 16 / 22 / 26 / 44`
- **圆角统一为 0**（极简 / 瑞士风）
- 视觉语言：`// 注释` 当章节标题，行间距紧凑

## 4 张设计稿对应页面

### ACN1 — TODAY（首页）
- 顶部：`// 05.18` 周一 ／ 右上 `第 21 周 / 2026·05`
- 标题卡："今日训练" / "推 训练日"
- Tag：`胸 肩 三头`
- 三栏统计：`05` 项目 / `18` 组数 / `45` 分钟（doto 大号字）
- 列表 `// 计划`：序号 `01..05` + 名称 + `N × M` + 右侧按钮（首项 `开始`，其余 `—`）
- CTA：`开始训练 ▸`
- 底部 Tab：`TODAY · WORK · STATS · ME`

### ACN2 — 训练中（从 TODAY 进入的子页面）
- 顶部：`进行中 · 00:24:11` + `⏸ 暂停`
- 项目导航：`◀  项目 01 / 05 · 杠铃卧推  ▶`
- 5 个组进度点
- 大数据：`重量 80 kg` ｜ `次数 08`（doto）
- `第 03 / 05 组  ·  PR · 85kg`
- 单组快填行：`✓ ✓ 本组 04 05`
- 组间休息：`PRESET · 1:30` / `01:30` / `开始`
- 时间长度滑块 + 预设 chip：`60s 90s 2:00 3:00 5:00 +`
- 自定义时长输入：`分 02 秒 30  [应用 ▸]`
- `// 已完成` 历史：`01 80kg 08次 ✓`
- 底部按钮：`+ 备注` ｜ `记录本组 ▸`

### ACN3 — STATS（数据）
- 顶部：`// 数据` / `五月 · 26`
- 三栏统计：`16` 次数 / `28.5` 容量(t) / `07` 连续(天)
- `// 训练日历` + `← 4月 · 5月 · 6月 →`
- 月日历表格（W18~W22，含训练强度色块）
- `// 周容量 (kg)` `Δ +12%` — 柱状图（W15~W21）
- `// 最近` 列表：`周六 17 / 拉 · 52 分 / 5.4t`

### ACN4 — ME（个人）
- 顶部：`// 个人` / `我`
- 用户卡片：头像 + `LV. 12` 徽章 / `刘 恺` / `会员 · 162 天` / `目标 · 增肌`
- `// 累计`：`142` 总训练次数 / `384` 总时长(h) / `1.42` 百万kg / `08` 历史 PR
- `// 目标`：
  - `卧推 · 100kg`  `85%`  ──── 进度条
  - `深蹲 · 140kg`  `62%`
  - `每周 4 练`     `75%`
- 设置列表：`器械 · 已存 5 ›` ／ `单位 · kg·cm ›` ／ `同步 · Healthkit ›` ／ `退出登录 ›`

## 缺失的页面（需自行设计，沿用同语言）

- **WORK 标签页**：原版的"训练库"管理。新设计未提供，需按同样的视觉语言（`// 训练库` 标题、列表、序号、`+ 新增 ▸` 按钮等）构造。
- **新增 / 编辑训练项目**：弹窗或全屏表单。
- **训练完成总结**：从训练中流程退出后的小结页面。
